# custom-drag-target
CustomDragTarget control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

* **Git dependency**

Link to git repository:

```
dependencies = [
  "custom-drag-target @ git+https://github.com/MyGithubAccount/custom-drag-target",
  "flet>=0.28.3",
]
```

* **PyPi dependency**  

If the package is published on pypi.org:

```
dependencies = [
  "custom-drag-target",
  "flet>=0.28.3",
]
```

Build your app:
```
flet build macos -v
```
